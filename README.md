# YSAssetIdx

A Parser of a certain anime game's asset index (WIP)

Current target version: 3.5

## Requirements

- Python 3

## How to run

- Clone this repository
```shell
git clone https://github.com/paimooon/YSAssetIdx
```
- Place `0000006f.bin`
- Run `main.py`
```shell
python main.py
```
